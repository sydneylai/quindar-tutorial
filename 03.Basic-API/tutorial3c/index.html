Welcome to Quindar REST API Web Services

To test Quindar REST API:

  • First REST API - to check if server has active heart-beats, click here
  • Second REST API - to read telemetry, click here
  • Third REST API - to write new telemetry position data point to MongoDB, use the utility 'curl'. Refer to the example below

To test database write API, you can download and install the utility 'curl', and execute the command 'curl -X POST -H "Content-type: application/json" -d '{"vehicleId":"IBEX","timestamp":1457640420,"x":236294.1956, "y":116196.8879,"z":-34379.67682,"vx":-0.675287,"vy":0.508343,"vz":0.434496}' http://localhost:3000/services/v1/position' from a command window (on Windows), or a terminal (on MacOS or Linux).
