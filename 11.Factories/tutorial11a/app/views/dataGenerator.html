This tool will generate one random telemetry data point, and write to our platform.audacy.space database server (MongoDB). Please click the icon below to push data changes to MongoDB.

Benefits - MongoDB is a multi-node database cluster, which is distributed in 3 different geographical region for fast database read and for resilience and high availability. Another advantage is the use of schema-less design that allows easy data structure changes without rebuilding the application or backend database. This will also ease descriptive (e.g. metrics) and predictive (e.g. trend prediction, anomaly prediction) data analytics.
